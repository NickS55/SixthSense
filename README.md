## How to Run Locally
1) install yarn 
    - the machine this was made on is macOS
    - installed using homebrew ($ brew install yarn)

2) add depencies to the project 
    - $ yarn install

3) run 
    - yarn run build

4) local host
    - http://localhost:3000/
    - http://localhost:3000/application
    - to change port from 3000 change the number in main.go
